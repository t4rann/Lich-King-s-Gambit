# Lich King’s Gambit

Turn-based tactical game created for SoloDevelopment Jam.

<img width="1920" height="1080" alt="Screenshot from the game" src="https://github.com/user-attachments/assets/2af594ef-86af-428a-bf9b-e04192600cc6" />

## Overview

Control undead units on a grid, engage in tactical combat, and use a fusion mechanic to strengthen your army during the match.

The project was developed in a game jam environment with focus on rapid gameplay iteration and system-driven design.

## Features

- Grid-based combat with deterministic movement
- Unit fusion progression system
- State-based enemy AI
- Custom level editor tool

## Tech

Unity, C#, ScriptableObject, Custom Editor Tools
